❓ TROUBLESHOOTING

MongoDB Connection Issues?
  → Check MONGODB_URI in .env
  → Ensure IP whitelist on MongoDB Atlas
  → Test: mongosh "{MONGODB_URI}"

OpenAI API Not Working?
  → Verify API key is correct
  → Check account has credits
  → Ensure proper permissions set

CORS Errors?
  → Update FRONTEND_URL in .env
  → Should match http://localhost:5173 (or your domain)

Port Already In Use?
  → Change PORT in .env or run: PORT=5001 npm run dev
